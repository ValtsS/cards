**Cards**

*Git:*

`git clone https://github.com/ValtsS/cards.git`
`git checkout module01`

*Install*:

`npm install`


*Build:*

`npm run build`

Run:

`npm run dev`

or

`npm run preview`



*Test coverage:*

`npm run coverage`
